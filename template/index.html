HELI-ENGINE

On premise server as a service

What is HELI-ENGINE?

Heli-engine is an on premise server based platform which has a collection of Geospatial data analysis algorithm that contains Geoprocessing APIs, 2D & 3D tilling APIs, raster, vector and lidar processing APIs along with data generation modules.

Features

Geoprocessing

Geoprocessing is a set of APIs used to perform spatial data processing, vector data processing and converting data file types. It is a collection of various APIs to process and analyse spatial data.

Tiling

Raster and vector tiling APIs are a set of web-accessible tiles through machine learning algortihms . Tiling feature calls and break the spatial data in smaller parts to make rendering, visualization and analysis of data faster.

Data Science

Data analysts, data scientists, and developers can use the Data Science feature which is a module to make business decisions using their own data. With data science APIs, user can do cleaning, merging, deleting and interpreting csv and geojson data.

Routing

With "Micro Routing Services" you can customize your route using customer data. Routing features helps you to reach your desired destination with the most optimized routes in less time . Manage your multiple deliveries in a sequence and save your time and money.
